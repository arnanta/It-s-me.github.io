Travel Log

This was a project I started in July 2020. This project is basically a website where we can add the places we visited.

The system is all about using mapbox api to fetch locations using longitude and latitude and marking the places we visited.

While zooming into the maps the size of the marker gets bigger and more prominent. The places can be identified very precisely.

One can add a place visited but he/she should be authorized, which means a password or API_KEY is needed to enter data.

The system works fine with all the dependancies installed.

Technologies:
  • - JavaScript
  • - NodeJs
  • - React JS
  • - Mongoose
  • - Mongo Atlas
  • - Mapbox SDK
  • - Json
  • - HTML,CSS
