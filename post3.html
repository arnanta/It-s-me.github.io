Xavotsav 2020 official Website

This was a project I started with my teammates in November 2019. This is the offical website of the annual cultural fest Xavotsav 2020 of St. Xavier's College(Kolkata).

The system is all about using HTML,CSS, JavaScript, full page Js, particles js and Angular.

Multiple pages are there in the website like- about, team, events, etc.

One can see the events list which will be happening for the three days with timings and details about the events.

The team section tells us all the people involved in successful completion of the project.

Technologies:
  • - JavaScript
  • - HTML
  • - CSS
  • - Bootstrap
  • - Material UI
  • - Full page Js
  • - particles JS
  • - SCSS
